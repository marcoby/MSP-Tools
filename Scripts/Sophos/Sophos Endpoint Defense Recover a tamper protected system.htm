    Sophos Endpoint Defense: Recover a tamper protected system

    KB-000036125Sep 16, 202120 people found this article helpful


    Important
    Sophos is retiring its on-premise products mentioned in this article on 20 July 2023. For product retirement details, see our retirement calendar.

    Overview

    This article describes how to recover a tamper-protected Windows system if the other methods to turn off tamper protection are not viable. 

    For instructions on recovering a tamper-protected Mac endpoint, contact Sophos support for further assistance.

    Endpoints managed by Sophos Central

    Recover tamper protection password of deleted endpoints and servers

    1. Sign in to Sophos Central.
    2. Go to Logs & Reports > Recover Tamper Protection passwords.
    3. Click View details to expand the password/s that has been set on the endpoint or server.

    Notes:

    • The password at the top of the list is the most recent and can be used to authenticate on the local endpoint or server, allowing access to the Settings and the option to turn off Tamper Protection.
    • The report will show endpoints and servers that have been deleted over the previous 90 days. To keep the data beyond the 90 day period, you can export the reports for your own records.
    We recommend using the various methods to turn off Tamper Protection on a Windows device as detailed in the knowledge base article Sophos Endpoint: How to disable Tamper Protection.
     

    Recover tamper protection password in the registry

    Do the following recovery steps if all other methods are not viable.
     

    For Core Agent 2.15.4 and later

    Windows 10 and Windows Server 2012, 2016, and 2019

    1. Go to Settings, Update & Security, Recovery under Advanced start-up click Restart now.


       
    2. On Choose an option, click Troubleshoot, then click Advanced options and Command Prompt:


       
    3. After the restart, select an administrative account to continue and enter the password.
    4. Open Command Prompt.
    5. Type C: and press Enter.
      • Note: Your Boot drive may differ to C. If so, use a command such as DiskPart and list volume to display the available volumes.
    6. Type cd Windows\System32\drivers and press Enter.
    7. Type ren SophosED.sys SophosED.sys.old and press Enter.
    8. Type exit and press Enter.
    9. Click Continue.
    10. Once back in Windows, open Registry Editor.
    11. Back up the registry.
    12. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos MCS Agent  and set the Value data of Start to 0x00000004
    13. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\SAVService  and set the Value data of Start to 0x00000004
    14. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos AutoUpdate Service  and set the Value data of Start to 0x00000004
    15. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos Endpoint Defense\TamperProtection\Services and under every subkey in this location set the Value data of Protected to 0.
      • Example:
        • Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos Endpoint Defense\TamperProtection\Services\SAVService and set the Value data of Protected to 0.
    16. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos Endpoint Defense\TamperProtection\Config and set the Value data of SAVEnabled and SEDEnabled to 0.
    17. Set the Value data of Enabled to 0 in the following:
      • 32-bit: HKEY_LOCAL_MACHINE\SOFTWARE\Sophos\SAVService\TamperProtection
      • 64-bit: HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Sophos\SAVService\TamperProtection
    18. Restart the endpoint or server to complete setting enhanced tamper protection to off.

    Windows 7 and Windows Server 2008 R2

    1. Turn on or restart the endpoint or server.
    2. Press F8 to open Advanced Boot Options.
    3. Select Repair Your Computer and press Enter.

    4. At the System Recovery Options screen, select a language and keyboard input method and click Next.
    5. Select a local administrative account to sign in to and click OK.
    6. Under System Recovery Options, click Command Prompt:


       
    7. Open Command Prompt with admin privilege.
    8. Type D: and press Enter.
      • Note: Your drive may differ to D. If so, use a command such as DiskPart and list volume to display the available volumes.
    9. Type cd Windows\System32\drivers and press Enter.
    10. Type ren SophosED.sys SophosED.sys.old and press Enter.
    11. Type exit and press Enter.
    12. Click Restart.
    13. Once back in Windows, open Registry Editor.
    14. Back up the registry.
    15. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos MCS Agent  and set the Value data of Start to 0x00000004
    16. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos Endpoint Defense\TamperProtection\Config and set the Value data of SAVEnabled and SEDEnabled to 0.
    17. Set the Value data of Enabled to 0 in the following:
      • 32-bit: HKEY_LOCAL_MACHINE\SOFTWARE\Sophos\SAVService\TamperProtection
      • 64-bit: HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Sophos\SAVService\TamperProtection
    18. Restart the endpoint or server in normal mode to complete setting enhanced tamper protection to off. 

    For Core Agent 2.10.8 and earlier

    1. Start the endpoint or server in Safe Mode and sign in as administrator.
    2. In Run, type services.msc , then click OK.
    3. Right-click the Sophos Anti-Virus service, then select Properties.
    4. Set the Startup type to Disabled, then click OK.
    5. In Run, type regedit.exe , then click OK.
    6. Back up the registry.
    7. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos MCS Agent  and set the Value data of Start to 0x00000004.
    8. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Sophos Endpoint Defense\TamperProtection\Config and set the Value data of SAVEnabled and SEDEnabled to 0.
    9. Set the Value data of Enabled to 0 in the following:
       
      • 32-bit: HKEY_LOCAL_MACHINE\SOFTWARE\Sophos\SAVService\TamperProtection
      • 64-bit: HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Sophos\SAVService\TamperProtection
       
    10. Restart the endpoint or server in normal mode to complete setting enhanced tamper protection to off.
